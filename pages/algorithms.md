## What is an algorithm?

> Algorithm is a set of finite, well-defined steps or instructions designed to solve a problem or perform a computation (🤓)
>
> - [geekforgeeks](https://www.geeksforgeeks.org/dsa/introduction-to-algorithms/)

<div class="flex justify-between">
    <div>
    <h4>Need for algorithms</h4>
    <p>Anything we write to solve a problem can be considered an algorithm, even if it's a simple for loop</p>
    <p>Some algorithms use different techniques to</p>
    <ul>
        <li>Break problems into smaller, manageable parts <i>(divide and conquer)</i></li>
        <li>Remember previous results to avoid repeating work
            <i>(dynamic programming <sub>(memoization/tabulation)</sub>) </i>
        </li>
        <li>Make step-by-step decisions for the best outcome <i>(greedy approach)</i></li>
        <li>Explore possibilities systematically <i>(backtracking / brute force search)</i></li>
        <li>Organize and process data efficiently <i>(sorting)</i></li>
    </ul>
